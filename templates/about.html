About the software

The software for this blog is created by Smart Viking, using these technologies:

  • Python (Flask) For serverside programming
  • Sqlite3 For database storage
  • Postmarkup For rendering bbcode
  • Captchator Captcha service
  • JSColor The color picker for the blog editor
  • jQuery To simplify Javascript
  • sbbeditor Buttons for inserting bbcode in textarea (Javascript)
